HTML

(Hypertext Markup Language )

Structure Of Html :

            <   !DOCTYPE html  >
                <   html  >
                    <   head  >
                      <   title  > title tame <   /title  >
                      <   /head  >
                      <   body  >
                      <   /body  >
                <   /html  >

< !DOCTYPE html> :- It defines that this document is an HTML document.

< html> :- This is the root element of an HTML page.

< head> :- This element contains metadata (data about data) about the HTML page.

< title> :- This element specifies a title for the HTMl page (which is shown in the browser's title bar or in the page's tab ).

< body> :- This element difines the document's bosy,and is a container for all the visible contents ,such as headings,paragraphs,images ,hyperlinks,tables,lists,etc.

Tags :- Tages are nothing but a keyword which is enclosed between the angular brackets '<> ' .

Types Of Tages :-

1.   Container tag / Opening tag And Closing tag
Syntax :- < tagname> content here < /tagname>

2.   SelfClosing tag / Empty tag / Void elements
Syntax :- < /tagname>

Container tags :-

These tags contains both start tag and end tag

  1. Heading tags :-

  < h1> Heading tag 1 < / h1>
  < h2> Heading tag 2 < / h2>
  < h3> Heading tag 3 < / h3>
  < h4> Heading tag 4 < / h4>
  < h5> Heading tag 5 < / h5>
  < h6> Heading tag 6 < / h6>

  2. Paragraph tag :- < P> is used to write paragraph in our web page.

  3. Center tag :- < center> is used to shift content from left to center.

  4. Bold tag :- < b> is used to display the text in bod format.

  5. Strike tag :- < s> is used to strike out the (it draws a line in the middle of the text ) .

  6. Mark tag :- < mark> is used to highlight the text by providing yellow background colour.

  7. Anchor tag :- < a> is used to link from one page to another.

Selfclosing tags :-

In this tags opening and closing tags are combined into one . These tags are used to indicates that there is no content between opening and closing tags.

  1. input tag :- < input /> is used to collect smaller amount of data from user.

  2. Breaking tag :- < br /> is used to move the cursor from current position to starting of next line.

  3. Horizontal Line :- < hr /> is used to draw a horizontal line on web browser.
